Срочно исправить верстку блока загрузки WB-отчётов.

Проблема:
сейчас на странице появились лишние нативные input[type=file] — строки “Выберите файл / Файл не выбран”. Также новый блок загрузки вложен поверх старого блока “Загрузка файла”, из-за чего получилась дублирующаяся и сломанная структура.

Что исправить:

1. Убрать нативные file input из видимой верстки

Не должно быть видно:

“Выберите файл”
“Файл не выбран”

input[type=file] должен быть скрыт:

display: none;

Загрузка должна запускаться только через кастомные кнопки:

“Загрузить отчёт”
“+ Добавить отчёт”
“Заменить”

2. Убрать старый блок “Загрузка файла”

Сейчас одновременно отображаются:
- новый блок “Еженедельные детализированные отчёты”
- старый блок “Загрузка файла”

Так быть не должно.

Оставить только новую структуру:

Еженедельные детализированные отчёты
Загрузите до 4 отчётов WB для объединённой аналитики.
[Загрузить отчёт]
Отчёты не загружены

И ниже отдельно:

Себестоимость товаров

3. Не вкладывать новый блок внутрь старого

Новая структура должна быть:

<Card>
  Еженедельные детализированные отчёты
  ...
</Card>

<Card>
  Себестоимость товаров
  ...
</Card>

А не:

<Card Загрузка файла>
  <Card Еженедельные отчёты>
  <Card Себестоимость>
</Card>

4. Кнопка “Скачать метрики в PDF”

Она не должна висеть внутри пустого блока загрузки.

Правильно:
- если отчёты не загружены — кнопка disabled или скрыта;
- расположить её в верхней строке блока отчётов справа, либо отдельно над аналитикой;
- не помещать её в старый блок “Загрузка файла”.

5. Пустое состояние

Когда отчётов нет, показывать только:

Еженедельные детализированные отчёты
Загрузите до 4 отчётов WB для объединённой аналитики.

[Загрузить отчёт]

Отчёты не загружены

Не показывать список слотов, не показывать два input file.

6. После загрузки отчёта

Показывать:

Загружено 1 из 4 отчётов

[20.04.2026 – 26.04.2026]
Еженедельный детализированный отчёт
filename.csv
[Заменить] [Удалить]

[+ Добавить отчёт]

Аналитика строится по отчёту 20.04.2026 – 26.04.2026

7. Итоговая структура страницы

Должно быть так:

Загрузка файла

  Еженедельные детализированные отчёты
  Загрузите до 4 отчётов WB для объединённой аналитики.
  [Загрузить отчёт / + Добавить отчёт]
  список загруженных отчётов или “Отчёты не загружены”

  Себестоимость товаров
  Добавьте файл себестоимости...
  [Загрузить свежий файл] [Удалить]
  Загружен: Себестоимость

Не должно быть:
- видимых “Выберите файл”
- старого пустого блока “Загрузка файла” внутри нового
- двух разных интерфейсов загрузки отчёта одновременно
- кнопки PDF внутри пустого file-card
